# Third-party notices

The base runtime uses only the Python standard library and contains no vendored third-party code.

Build and development tools are installed separately and are not redistributed in project
artifacts: Hatchling (MIT), build (MIT), pytest (MIT), Hypothesis (MPL-2.0), Ruff (MIT), and mypy
(MIT). The continuous-integration workflow uses official checkout and Python setup actions under
their respective repository licenses.

This inventory is informational and should be reviewed when dependencies change.
