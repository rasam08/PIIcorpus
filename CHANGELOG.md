## 0.1.0 - 2026-07-22

- Published the source repository, documentation, and first GitHub release.
- Added deterministic generation, independent validation, failure-mode auditing, strict annotated
  text import, and five span-preserving exporters.
- Added fictional demo data, deliberately defective audit cases, documentation, packaging, and CI.
- Removed the label-exclusive hard-negative index marker and replaced it with a class-balanced
  synthetic context/reference suffix.
- Added explicit cue-to-entity links, per-split cue-free checks, and emitted-evidence validation
  for cue/shape contrastive records.
- Made audit and export strict-validation consumers by default, with a failure-preserving forensic
  override for invalid inputs.
- Added content-derived case-ID verification, literal metadata-surface checks, kind-predictive
  marker auditing, and regression corpora for the external audit exploits.
- Extended kind-marker auditing across one-, two-, and three-token features, and made cue-shortcut
  ceilings independently enforceable in every split.
- Added the editable-build helper to the reviewed development lock so fresh no-isolation installs
  follow the documented CI path.
- Pinned the CI runner and reviewed development dependency lock.
- Published release-date citation metadata for the tagged `v0.1.0` release.
- Forced LF checkout semantics for JSON and JSONL corpus files and included the rule in source
  distributions.

No package has been published to a package index.
